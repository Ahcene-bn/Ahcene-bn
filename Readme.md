### Echo Reborn

Jeu de rôle au tour par tour développé avec **MonoGame**. Le joueur contrôle un ou plusieurs personnages, explore un monde composé de plusieurs lieux et participe à des combats au tour par tour contre des ennemis.

Le projet comprend :

* Modélisation UML
* Développement en C#
* Framework .NET
* Traitement de données XML

https://github.com/d0rich/EchoReborn
